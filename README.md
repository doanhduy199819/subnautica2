# Subnautica 2 - Save Game Restore Guide

If you need to restore your Subnautica 2 save files (due to a PC transition, a game crash, or backing up a multiplayer world), follow the directory paths and instructions below.

---

## 📂 Save File Locations

Depending on where you purchased/play the game, your saves are located in different directories.

### 1. LND
Find saved game directory at:
# <your_disk>:\Users\<your_user_profile>\AppData\Local\Subnautica2\Saved\SaveGames
